# Breakpoint Media Queries

Synergy user [custom media queries](https://github.com/csstools/postcss-plugins/tree/main/plugins/postcss-custom-media) in CSS following the [Custom Media Specification](https://www.w3.org/TR/mediaqueries-5/#at-ruledef-custom-media).

The media queries available may depend on the brand.

> Due to limitation with CSS variables in Custom Media Queries these values are hardcoded and need to be managed manually.

```css
@custom-media --sds-screen-xs (max-width: 400px);
@custom-media --sds-screen-sm (min-width: 576px);
@custom-media --sds-screen-md (min-width: 768px);
@custom-media --sds-screen-lg (min-width: 992px);
@custom-media --sds-screen-xl (min-width: 1200px);
@custom-media --sds-screen-xxl (min-width: 1440px);
```

## Usage

```css
@media (--sds-screen-md) {
	/* styles for medium viewport */
}
```

Media queries can also be nested using PostCSS. In the example below the `section` selector uses the mobile padding on small screen but the desktop padding on the medium screens (or higher).
```css
.section {
	padding: var(--sds-semantic-spacing-section-mobile-padding-y) var(--sds-semantic-spacing-section-mobile-padding-x);

	@media (--sds-screen-sm) {
		padding: var(--sds-semantic-spacing-section-desktop-padding-y) var(--sds-semantic-spacing-section-desktop-padding-x);
	}
}
```

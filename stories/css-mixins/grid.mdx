# Grid

Synergy DNA provides helper mixins to simplify working with responsive CSS Grids.

## Creating grids

The `grid-cols` mixin can be used to create a CSS grid with the provided number of columns. 

```css
@mixin grid-cols {columns};
```

## Grid Column Span

The `grid-col-span` mixin can be used to span grid columns.

<Story of={GridStories.SpanColumns}/>

```html
<div class="grid-demo-3">
    <div>col (span 3)</div>
    <div>col (span 9)</div>
    <div>col (span 3)</div>
    <div>col (span 9)</div>
</div>
```

```css
.grid-demo-3 {
    display: grid;
    @mixin grid-cols 12;

    & > div {
        &:nth-child(even) {
            @mixin grid-col-span 9;
        }

        &:nth-child(odd) {
            @mixin grid-col-span 3;
        }
    }
}
```

### Responsive Column Span

The column span can be adjusted by breakpoint using the responsive versions of the column span mixins.

```css
@mixin grid-{breakpoint}-cols {columns};
```

## Grid Row Span

The `grid-col-span` mixin can be used to span grid row.

<Story of={GridStories.SpanRows}/>

```html
<div class="grid-demo-4">
    <div>col (row 3)</div>
    <div>col (span 9)</div>
    <div>col (span 9)</div>
    <div>col (span 9)</div>
</div>
```

```css
.grid-demo-4 {
    display: grid;
    @mixin grid-cols 12;

    & > div {
        &:first-of-type {
            @mixin grid-row-span 3;
        }

        &:not(:first-of-type) {
            @mixin grid-col-span 11;
        }
    }
}
```

The column span can be adjusted by breakpoint using the responsive versions of the row span mixins.

## Gutter

The `gutter` mixin can be used to add spacing between grid columns;

<Story of={GridStories.SpanColumns}/>

```html
<div class="grid-demo-5">
    <div>col (span 3)</div>
    <div>col (span 9)</div>
</div>
```

```css
.grid-demo-5 {
    display: grid;
    @mixin grid-cols 12;

    & > div {
        @mixin grid-col-span 9;

        &:first-of-type {
            @mixin grid-col-span 3;
        }
    }
}
```

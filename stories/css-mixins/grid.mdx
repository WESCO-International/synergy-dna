# Grid

Synergy DNA provides helper mixins to simplify working with responsive CSS Grids.

<Story of={GridStories.Demo}/>

## Creating grids

The `grid-cols` mixin can be used to create a CSS grid with the provided number of columns. 

```css
@mixin grid-cols {columns};
```

### Responsive Grids

The `grid-cols` can be adjusted by breakpoint using the responsive versions of the `grid-cols` mixins.

```css
@mixin grid-{breakpoint}-cols {columns};
```

### Example with 8 responsive columns

The demo below shows how to implement as responsive grid that is 8 columns on large screens, 4 columns on medium screens, 2 columns on small screens and 1 column for everything else.

<Story of={GridStories.ResponsiveColumns}/>

```html
<div class="grid-demo-responsive">
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
</div>
```

```css
.grid-demo-responsive {
    display: grid;
    @mixin grid-cols 1;
    @mixin grid-sm-cols 2;
    @mixin grid-md-cols 4;
    @mixin grid-lg-cols 8;
    @mixin gap xs;
    & > div {
        @mixin layer-1;
        text-align: center;
    }
}
```

## Grid Column Span

The `grid-col-span` mixin can be used to span grid columns.

<Story of={GridStories.SpanColumns}/>

```html
<div class="grid-demo-3">
    <div>col (span 3)</div>
    <div>col (span 9)</div>
    <div>col (span 3)</div>
    <div>col (span 9)</div>
</div>
```

```css
.grid-demo-3 {
    display: grid;
    @mixin grid-cols 12;

    & > div {
        &:nth-child(even) {
            @mixin grid-col-span 9;
        }

        &:nth-child(odd) {
            @mixin grid-col-span 3;
        }
    }
}
```

### Responsive Column Span

The column span can be adjusted by breakpoint using the responsive versions of the column span mixins.

## Grid Row Span

The `grid-row-span` mixin can be used to span grid row.

<Story of={GridStories.SpanRows}/>

```html
<div class="grid-demo-4">
    <div>col (row 3)</div>
    <div>col (span 9)</div>
    <div>col (span 9)</div>
    <div>col (span 9)</div>
</div>
```

```css
.grid-demo-4 {
    display: grid;
    @mixin grid-cols 12;

    & > div {
        &:first-of-type {
            @mixin grid-row-span 3;
        }

        &:not(:first-of-type) {
            @mixin grid-col-span 11;
        }
    }
}
```

### Responsive Row Span

The column span can be adjusted by breakpoint using the responsive versions of the row span mixins.

```css
@mixin grid-{breakpoint}-row-span {rows};
```

## Gap

The `gap` mixin can be used to add uniform spacing between grid columns and rows.

<Story of={GridStories.Gap}/>

```html
<div class="grid-demo-5">
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
    <div>col</div>
</div>
```

```css
.grid-demo-5 {
    display: grid;
    @mixin grid-cols 4;
    @mixin gap 2xl;

    & > div {
        @mixin layer-1;
        text-align: center;
    }
}
```

# grid-auto-flow-row

The `grid-auto-flow-row` mixin can be used to control how the auto-placement algorithm works, specifying exactly how auto-placed items get flowed into the grid. It is functionally equivelent to setting `grid-auto-flow:row`. It is provided as part of Synergy as convience when working with the responsive versions.

```css
@mixin grid-auto-flow-row;
```

### Responsive Grid Auto Flow Row

The `grid-auto-flow-row` mixin can be adjusted by breakpoint using the responsive versions.

```css
@mixin grid-{breakpoint}-auto-flow-row;
```

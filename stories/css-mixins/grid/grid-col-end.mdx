# grid-col-end

The `grid-col-end` mixin can be used to specifies a grid items end position within the grid.

```css
@mixin grid-col-end {position};
```

### Responsive Grid Column End

The `grid-col-end` mixin can be adjusted by breakpoint using the responsive version.

```css
@mixin grid-{breakpoint}-col-end {position};
```

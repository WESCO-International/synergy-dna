# CSS Utility Classes for Shadows

Synergy provides the following CSS Utility classes to work with shadows.

### Demo
<Story of={ShadowStories.Small}/>

#### Code Example
```html
<div>
    <div class="shadow-sm">shadow-sm</div>
</div>
```

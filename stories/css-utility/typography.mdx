# CSS Utility Classes for Typography

Synergy provides a set of CSS Utility Classes for working with type.

## Font Size Utilities

Keeping the correct semantic ordering of headings on a page is important for accessibility. The sequencing of headings impacts the navigation of the content when accessed through a screen reader. Incorrect ordering of headings can cause difficulties in comprehending the text for those using a screen reader.

For example you should never have an `h1` followed by an `h3` and skip `h2`.

There are situations though where you want the semantic ordering of headings to be correct but visually you may want the text to be a different size. For these cases, Synergy provides some CCS Utility classes that allow you to render in a different size.

### Demo
<Story of={TypographyStories.FontSizeUtilities}/>

#### Code Example
```html
<div>
  <h1>This is an h1</h1>
  <h2 class="h4">This is an h2, with the size of an h4</h2>
</div>
```

## Font Weight Utilities

Synergy provides a set of utility classes for working with font weights. Font weight handling is handled by default by the system. So if a piece of text is bolded by an author in Word this will carry through on the front end and will use the correct bold font weight for the brand. 

There may be situations though where a developer wants to change the font weight by decorating a `div` or `span` in the dom. The font weight utility classes can be used for this.

### Demo
<Story of={TypographyStories.FontWeightUtilities}/>

#### Code Example
```html
<div>
  <h3>This is an <span class="bold">h3</span></h3>
</div>
```

### Supported Classes

The follow CSS Utility classes are supported by Synergy.

> The `semi-bold` class is currently only supported on text within a `p`.

  <div className="row">

  </div>

## Text Alignment Utilities

Synergy provides a set of utility classes for working with text alignment. 

Developers can add these classes to DOM elements or authors could add these classes to block names to control text alignment.

### Demo
<Story of={TypographyStories.TextAlignmentUtilities}/>

#### Code Example
```html
<div>
  <div class="text-align-left">Let Aligned</div>
  <div class="text-align-center">Center Aligned</div>
  <div class="text-align-right">Right Aligned</div>
</div>
```
